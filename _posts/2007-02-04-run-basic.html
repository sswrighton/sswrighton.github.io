--- layout: post title: Run BASIC date: '2007-02-04T20:37:00.000-06:00' updated: "2019-03-03 19:46:00" permalink: "/2007/02/run-basic.html" author: Stephen Wrighton tags: - Link modified_time: '2007-02-04T20:53:22.715-06:00' blogger_id: tag:blogger.com,1999:blog-947968907697915479.post-494862744082210467 blogger_orig_url: https://programmer.wrighton.org/2007/02/run-basic.html --- A week or so ago, I read an essay from a rather well-known science-fiction writer complaining about the fact that he was unable to find a simple, free and easy-to-use BASIC compiler which he could use with his son to build those sample exercises that grammar school math books always seems to have at the end of the chapter.

Enter Run BASIC.

It's a web-based BASIC compiler. I'm talking the old-style BASIC, which I learned how to code in on a Commodore 64 way back in the day. Useful? Probably not that much, but it sure is fun!
