--- layout: post title: Google Docs date: '2007-06-27T07:54:00.001-05:00' author: Stephen Wrighton tags: - User Interface - Web modified_time: '2007-06-27T07:58:36.173-05:00' thumbnail: http://bp1.blogger.com/_0dC8-RtLatg/RoJeKC8dWPI/AAAAAAAAACg/LvX1PrVl2FM/s72-c/gogdoc.png blogger_id: tag:blogger.com,1999:blog-947968907697915479.post-437540987734775151 blogger_orig_url: https://programmer.wrighton.org/2007/06/google-docs.html --- Well, I logged into Google Docs this morning (much as I do every morning, it's one of my STARTUP Tabs in Firefox) and got a nice surprise. A brand new layout for the file system.
Frankly, I like it a lot better than the old layout. I like the folders, and the "Outlook" style that it is using here. It's more user friendly, and more intuitive to use-after all it is the paradigm that is used by Windows File Explorer, Outlook, and Google's own GMail and Reader applications.

I for one will not miss the pull down menus that were up there, and am glad they've made these changes. Thanks, Google.
